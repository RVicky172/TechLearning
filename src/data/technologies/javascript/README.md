# JavaScript Technology Data

This folder contains the learning content for the JavaScript track. It focuses on core language behavior, functions, asynchronous programming, modern language features, and the browser DOM.

## Current Structure

```text
javascript/
├── README.md
├── index.ts
├── core.ts
├── functions.ts
├── async.ts
├── modern.ts
└── dom.ts
```

## Section Inventory

Top-level order in `javascript/index.ts`:

1. Core JavaScript
2. Functions
3. Async JavaScript
4. Modern JavaScript
5. DOM and Browser APIs

## How It Works

- Each file exports one named `TopicNode`.
- `javascript/index.ts` assembles the final `Technology` object.
- Shared types are defined in `src/data/types.ts`.
